# ar-mine.github.io

## My academic page.

### Main branch saves the files generating hexo web pages, while gh-pages branch is the actual codes for academic pages running and images branch is used as images' bed.

## Some Plugin and themes used
```
$ cd resume-blog/themes
$ git clone https://github.com/AngryPowman/hexo-theme-prontera themes/prontera

# Jade Plugin and its template
$ npm install hexo-renderer-jade --save
$ npm install hexo-generator-feed --save

# Plugin for Github Deployment
$ npm install hexo-deployer-git --save

# Plugin for top the article
$ npm install hexo-generator-index-pin-top --save

# Plugin for insert Youtube Player
$ npm install hexo-tag-youtube-responsive --save

# Plugin for hidden articles
$ npm install hexo-hide-posts --save
```
